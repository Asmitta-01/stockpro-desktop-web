Installation

  1. Téléchargez le fichier d'installation en cliquant sur "Télécharger".
  2. Double-cliquez sur le fichier téléchargé pour lancer l'installateur.
  3. Suivez les invites de l'assistant d'installation.
  4. Ouvrez StockPro depuis le raccourci créé sur le bureau.

Aucune connexion Internet n'est requise après l'installation. Pour le mode LAN, connectez vos postes sur le même réseau local.
